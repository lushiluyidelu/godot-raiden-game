## 游戏参数当前值

```gdscript
# 玩家
PLAYER_SPEED = 300
PLAYER_MAX_HEALTH = 3
PLAYER_INVINCIBLE_DURATION = 1.5

# 武器系统
WEAPON_LEVEL_MAX = 4
WEAPON_XP_PER_LEVEL = [3, 6, 9, 12]

# 道具
POWERUP_DROP_RATE = 0.15  # 15%

# 连击
COMBO_TIMEOUT = 3.0  # 秒
COMBO_MULTIPLIER = {
    2: 1.2,
    5: 1.5,
    10: 2.0,
    20: 3.0
}

# 关卡
TOTAL_LEVELS = 5
```

## 控制说明

| 操作 | 按键 |
|------|------|
| 向上移动 | W / ↑ |
| 向下移动 | S / ↓ |
| 向左移动 | A / ← |
| 向右移动 | D / → |
| 射击 | 空格键 / 鼠标左键 |
| 使用炸弹 | U 键 |
| 重新开始 | 回车键（游戏结束后） |
