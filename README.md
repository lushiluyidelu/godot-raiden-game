# 雷电游戏 (Thunder Raid)

> 一个使用 Godot 4.x 开发的 2D 垂直滚动射击游戏

## 🎮 项目简介

这是一个类似经典雷电游戏的射击游戏，玩家控制一架飞机，射击从屏幕上方出现的敌机，获得分数。

**技术栈**:
- 引擎：Godot 4.x
- 语言：GDScript 2.0
- 分辨率：480x720 (竖版)

## 🚀 快速开始

### 环境要求

- Godot 4.x (推荐最新稳定版)
- Windows/macOS/Linux

### 运行游戏

1. 打开 Godot Engine
2. 导入项目 (`project.godot`)
3. 按 `F5` 运行游戏

### 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格键 | 射击 |
| 回车键 | 重新开始 (游戏结束后) |

## 🛠️ 开发笔记

### 碰撞层配置

```
Layer 1: player
Layer 2: bullets
Layer 3: enemies
```

### 音效实现

使用 `AudioStreamWAV` 程序化生成音频：
- 采样率：22050 Hz
- 格式：16-bit PCM
- 数据：`PackedByteArray`
