# 雷电游戏 - 产品需求文档 (PRD)

## 项目概述

| 项目 | 描述 |
|------|------|
| **项目名称** | 雷电游戏 (Thunder Raid) |
| **游戏类型** | 2D 垂直滚动射击游戏 (STG) |
| **目标平台** | PC (Windows/macOS/Linux) |
| **游戏引擎** | Godot 4.x |
| **脚本语言** | GDScript |
| **目标受众** | 射击游戏爱好者、休闲玩家、Godot 学习者 |

---

## 用户故事

### 核心玩家故事
1. **作为一个玩家**，我想要控制一架飞机在屏幕上移动，以便躲避敌人和子弹
2. **作为一个玩家**，我想要发射子弹击毁敌人，以便获得分数
3. **作为一个玩家**，我想要看到敌机从屏幕上方出现，以便有目标可以射击
4. **作为一个玩家**，我想要在击毁敌人时听到爆炸音效，以便有打击感
5. **作为一个玩家**，我想要看到爆炸特效，以便视觉反馈更明显

### 游戏进度故事
6. **作为一个玩家**，我想要有生命值系统，以便可以承受几次失误
7. **作为一个玩家**，我想要看到游戏结束界面，以便知道我的最终分数
8. **作为一个玩家**，我想要能够重新开始游戏，以便快速再玩一次

### 视觉/听觉故事
9. **作为一个玩家**，我想要看到漂亮的飞机图形，而不是简单的几何形状
10. **作为一个玩家**，我想要有星空背景滚动效果，以便有飞行感
11. **作为一个玩家**，我想要有背景音乐，以便游戏氛围更好

## 功能需求

### 必须实现 (P0 - 核心功能)

| ID | 功能 | 描述 | 完成标准 |
|----|------|------|----------|
| F001 | 玩家控制 | WASD/方向键控制飞机移动 | 飞机可以 8 方向移动，不超出屏幕边界 |
| F002 | 射击系统 | 空格键发射子弹 | 按空格发射子弹，有射击间隔限制 |
| F003 | 敌人生成 | 敌人定时从屏幕上方生成 | 敌人以固定频率生成，水平移动 |
| F004 | 碰撞检测 | 子弹击中敌人、敌人撞到玩家 | 子弹消灭敌人，玩家撞到敌人扣血 |
| F005 | 分数系统 | 击毁敌人获得分数 | 左上角显示当前分数 |
| F006 | 生命值系统 | 玩家有 3 条命 | 撞到敌人扣血，血量为 0 游戏结束 |
| F007 | 游戏结束 | 显示游戏结束界面 | 显示最终分数，可以按回车重新开始 |

### 应该实现 (P1 - 重要功能)

| ID | 功能 | 描述 | 完成标准 |
|----|------|------|----------|
| F008 | 视觉改进 | 飞机有详细的图形 | 玩家和敌机有三角形机身和机翼 |
| F009 | 星空背景 | 滚动的星空背景 | 星星向下移动，营造飞行感 |
| F010 | 射击音效 | 发射子弹时的音效 | 每次射击有"咻咻"声 |
| F011 | 爆炸音效 | 敌人被击毁时的音效 | 击毁敌人有低沉爆炸声 |
| F012 | 背景音乐 | 游戏背景音乐 | 循环播放的电子音乐 |

### 可以实现 (P2 - 额外功能)

| ID | 功能 | 描述 | 完成标准 |
|----|------|------|----------|
| F013 | 爆炸特效 | 粒子爆炸效果 | 敌人死亡时有碎片飞散 |
| F014 | 多种敌人 | 不同颜色/速度/血量的敌人 | 至少有 2-3 种敌人类型 |
| F015 | 敌人子弹 | 敌人也会发射子弹 | 玩家需要躲避敌方子弹 |
| F016 | Boss 战 | 定时出现大 Boss | Boss 有多种攻击模式 |
| F017 | 道具系统 | 掉落增强道具 | 双发、加速、护盾等 |
| F018 | 高分记录 | 保存最高分 | 游戏结束后显示历史最高分 |

## 当前范围 (In Scope)

✅ **已实现的功能：**
- 玩家飞机控制（WASD/方向键）
- 射击系统（空格键）
- 敌人生成和移动
- 碰撞检测（子弹击毁敌人、敌人撞玩家）
- 分数系统
- 视觉改进（三角形飞机图形，程序化绘制）
- 星空背景滚动
- 音效系统（射击音效、爆炸音效）
- 背景音乐（mp3 文件加载，循环播放）
- 生命值系统（3 点生命，无敌时间 1.5 秒）
- 游戏结束界面（显示分数，回车键重新开始）
- 爆炸粒子特效
- 多种敌人类型（普通/快速/重型）

🔄 **进行中：**
- 无（核心功能已全部完成）

⏸️ **暂不实现（未来版本）：**
- 敌人发射子弹
- Boss 战
- 道具系统
- 多人模式
- 关卡系统
- 主菜单界面
- 高分记录

---

## 非目标 (Out of Scope)

以下功能明确**不包含**在当前版本中：

1. **多人游戏** - 不支持本地或在线多人
2. **关卡编辑器** - 不提供自定义关卡功能
3. **成就系统** - 没有 Steam 成就或其他成就
4. **内购/微交易** - 完全免费，无任何付费内容
5. **移动端适配** - 仅针对 PC 平台，不做触屏适配
6. **剧情模式** - 没有复杂的剧情或对话
7. **飞机升级系统** - 没有 RPG 元素

## 技术规范

### 性能目标
- **帧率**: 稳定 60 FPS
- **分辨率**: 480x720 (竖版)
- **加载时间**: < 2 秒

### 兼容性
- **Godot 版本**: 4.x (最新稳定版)
- **操作系统**: Windows 10+, macOS 10.15+, Linux

### 代码规范
- 使用 GDScript 2.0
- 遵循 Godot 官方代码风格指南
- 所有脚本使用英文注释
- 场景文件使用 .tscn 格式
