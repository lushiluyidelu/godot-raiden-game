## 视觉特效

### 星空背景实现

```gdscript
# 星星数据结构
var stars = []
for i in range(100):
    stars.append({
        "pos": Vector2(randf() * 480, randf() * 720),
        "speed": randf_range(20.0, 60.0),
        "size": randf_range(1.0, 2.5),
        "brightness": randf_range(0.5, 1.0)
    })
```

### 绘制星星

```gdscript
func _draw():
    for star in stars:
        var color = Color(star.brightness, star.brightness, star.brightness + 0.2, 0.8)
        draw_circle(star.pos, star.size, color)
```

## 程序化图形绘制

### 玩家飞机绘制要点

```
1. 创建 64x64 透明图像
2. 绘制三角形机身（渐变蓝色）
3. 绘制左右机翼
4. 绘制驾驶舱（浅蓝色）
5. 绘制引擎火焰（橙色，随机高度）
```

### 敌人飞机绘制要点

```
1. 创建 48x48 透明图像
2. 绘制倒三角形机身（红色渐变）
3. 绘制左右机翼（深红色）
4. 绘制驾驶舱（暗红色）
```

### 子弹绘制要点

```
1. 创建 16x32 透明图像
2. 绘制长条形主体（黄色到橙色渐变）
3. 绘制核心（亮白色）
```

## 性能考虑

### 绘制优化

- 使用程序化图形，不需要外部图片资源
- 纹理在 `_ready()` 时创建一次
- `_draw()` 只绘制简单几何图形

### 内存管理

- 及时删除屏幕外的子弹和敌人
- 使用 `queue_free()` 释放资源
